The experiments.zip can be downloaded from:
https://drive.google.com/file/d/1ey6AtwBQ2dkRKitoxNMpdnl4FZ2CXOeN/view?usp=sharing

Before replicating the experiments, ensure you have the following:
* A CSV file containing the list of CVEs and their corresponding NVD Disclosure date.
* DATA directory containing the vulnerable artifacts
* NON_VULN_DATA directory containing the non-vulnerable artifacts
* GITHUB_ARCHIVE_DATA directoy containing the GITHUB ARCHIVE data.
